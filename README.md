# 🎬 MoviePulse - Streaming Platform

A production-grade, mobile-first PWA streaming platform powered by Firebase.

## 🌐 Live Site
**https://moviepulsemedia.vercel.app**

## ✨ Features
- 🎥 Netflix-style streaming experience
- 🔐 Firebase Auth (Email/Password + Google)
- 💳 Subscription plans (Daily/Weekly/Monthly/Lifetime)
- 📱 Mobile-first PWA with offline support
- ⭐ 5-star rating system
- 📤 Social sharing (WhatsApp, Facebook, Twitter, Telegram)
- 🔥 Live "X watching now" counters
- 📅 Daily streak rewards
- 🌙 Dark/Light mode toggle

## 🚀 Deployment
This project is deployed on Vercel at: **https://moviepulsemedia.vercel.app**

## 🔧 Firebase Setup
1. Create Firebase project
2. Enable Authentication (Email/Password + Google)
3. Create Firestore collections: `movies`, `users`, `settings`
4. Add your first movie document

## 📁 Tech Stack
- HTML/CSS/JavaScript
- Firebase (Auth, Firestore)
- Vercel (Hosting)
